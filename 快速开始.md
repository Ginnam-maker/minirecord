# 快速开始指南

## 第一步：安装 Pinia

在项目根目录运行：

```bash
npm install pinia
```

或在 HBuilderX 中：
1. 右键点击项目根目录
2. 选择"使用命令行窗口打开所在目录"
3. 运行 `npm install pinia`

## 第二步：配置 DeepSeek API Key

1. 访问 [DeepSeek 开放平台](https://platform.deepseek.com/)
2. 注册并登录账号
3. 创建 API Key
4. 打开项目中的 `config.js` 文件
5. 将你的 API Key 填入 `deepseek.apiKey` 字段：

```javascript
export default {
  deepseek: {
    apiKey: 'sk-xxxxxxxxxxxxxxxx', // 替换为你的 API Key
    // ...
  }
}
```

## 第三步：准备底部标签栏图标（可选）

### 方案一：使用图片图标（推荐）

在 `static/tabbar/` 目录下放置以下图标文件（81x81 px）：
- `record.png` - 记录图标（未选中）
- `record-active.png` - 记录图标（选中）
- `summary.png` - 总结图标（未选中）
- `summary-active.png` - 总结图标（选中）

**图标资源获取：**
- [iconfont 阿里图标库](https://www.iconfont.cn/)
- [iconify 图标库](https://icon-sets.iconify.design/)
- [Flaticon](https://www.flaticon.com/)

推荐搜索关键词：
- 记录页：notebook, edit, calendar, pencil
- 总结页：chart, analytics, report, document

### 方案二：临时去除图标（快速测试）

如果暂时没有图标，可以先注释掉图标配置：

打开 `pages.json`，找到 tabBar 配置，注释掉 iconPath 相关行：

```json
{
  "tabBar": {
    "list": [
      {
        "pagePath": "pages/record/list",
        // "iconPath": "static/tabbar/record.png",
        // "selectedIconPath": "static/tabbar/record-active.png",
        "text": "记录"
      },
      {
        "pagePath": "pages/summary/index",
        // "iconPath": "static/tabbar/summary.png",
        // "selectedIconPath": "static/tabbar/summary-active.png",
        "text": "总结"
      }
    ]
  }
}
```

这样标签栏会只显示文字，不显示图标。

## 第四步：运行项目

### 在 HBuilderX 中运行：

1. 打开 HBuilderX
2. 导入项目（文件 -> 导入 -> 从本地目录导入）
3. 选择运行平台：
   - **H5**：运行 -> 运行到浏览器 -> Chrome
   - **微信小程序**：运行 -> 运行到小程序模拟器 -> 微信开发者工具
   - **App**：运行 -> 运行到手机或模拟器

### 首次运行检查：

- ✅ 底部标签栏显示正常
- ✅ 可以切换"记录"和"总结"页面
- ✅ 日历组件显示正常
- ✅ 可以输入并保存记录

## 使用流程

### 1. 添加每日记录

1. 打开"记录"页面
2. 在日历中选择日期（默认今天）
3. 在输入框中写下今天做了什么
4. 点击"保存记录"按钮

### 2. 生成周总结

当你记录了一周的内容后：

1. 切换到"总结"页面
2. 确认周期（本周/上周）
3. 点击"生成总结"按钮
4. 等待 AI 生成总结（约5-10秒）
5. 查看生成的总结内容

### 3. 查看历史

- **记录历史**：在记录页面向下滚动查看历史记录
- **总结历史**：在总结页面查看之前生成的所有总结

## 常见问题

### 问：运行时提示 "pinia is not defined"

答：需要先安装 pinia 依赖，运行 `npm install pinia`

### 问：底部标签栏不显示

答：检查 `pages.json` 中 tabBar 配置是否正确，图标文件是否存在

### 问：生成总结时提示 API Key 错误

答：检查 `config.js` 中的 API Key 是否正确填写，确保 Key 有效且有足够额度

### 问：日历组件不显示

答：确保项目中已包含 uni-ui 组件库（项目模板已包含）

### 问：如何清空所有数据？

答：在浏览器中按 F12，在控制台运行：
```javascript
localStorage.clear()
```
或在小程序中，长按应用图标 -> 删除数据
